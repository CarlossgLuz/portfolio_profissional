# Security Policy

## Scope

This repository is public, but any fraudulent reuse of Carlos Gabriel's identity, official links, resume or portfolio presentation should be treated as a security issue.

## Report

If you find:

- phishing pages copying this portfolio
- impersonation using Carlos Gabriel's name or links
- malicious mirrors pretending to be official
- misleading reuse of the resume or certification references

please report it with:

- affected URL
- date/time observed
- screenshots or archived evidence when possible
- a short description of the issue
