## **Descrição**

Este repositório contém o código do meu **Portfolio Oficial**, desenvolvido para ser uma vitrine profissional clara, minimalista e objetiva.  
O projeto oferece:

- Apresentação profissional  
- Mostruário de projetos  
- Sessão de contato  
- Layout responsivo otimizado  
- Identidade visual consistente  

O foco é demonstrar domínio em **UI/UX**, **Front-End**, organização de código e boas práticas de arquitetura visual.

O portfolio está em **melhoria contínua**: novos projetos, melhorias de interface e novas animações serão adicionados regularmente.

## **Funcionalidades**

### Estrutura Principal
- Hero minimalista com foco em identidade visual  
- Sessão "Sobre"  
- Sessão "Projetos"  
- Sessão "Contato" alinhada ao formulário  
- Rodapé organizado  

### UI/UX
- Layout limpo e responsivo  
- Organização por seções independentes  
- Paleta suave com contraste adequado  
- Botão de "Voltar ao Topo"  

### Responsividade
- Tamanhos específicos para mobile    
- Header completamente funcional em todas resoluções  
- Menu hambúrguer

## **Instalação**

```bash
# 1. Clone o repositório
git clone https://github.com/CarlossgLuz/portfolio_profissional.git

# 2. Acesse o diretório
cd portfolio_profissional

# 3. Execute abrindo o arquivo principal
open index.html
```

Windows:

```bash
start index.html
```
